About me

Hi, I'm Siv - a Data Analyst / Project Coordinator with a solid foundation in data analytics, programming, stakeholder management, and asset documentation, developed throughout my degree in Actuarial Science, work experience and personal endeavours.

I have been working in the Australian Defence sector, assisting the HOTO (Handover-Takeover) process for large-scale construction projects for the last three years.
This has given me the opportunity to not only develop my technical skills, but has considerably developed my own knowledge base and communication skills, enabling me to work effectively with cross-functional teams.

With a flexible skillset focused around Python and Excel and with proficiency using SQL, Alteryx and BI tools like Power BI and Tableau, I am well equipped to tackle complex data problems, streamline workflows, optimize business processes and inform data-driven decision making.

Projects

Automated Budget Tracking
  • Github

A personalised multi-page Power BI dashboard to visualise expenses and for budgeting purposes.
Python has been used to extract, transform then load the data to a database which can be used in Power BI. The Selenium package is used to automate the logging in and extraction of transaction data while the Pandas package formats the data to create a general, useable database. User inputs have also been employed where required using the Tkinter Package.
Code can be viewed above and due to the personal information in the data, access to the dashboard has not been provided.

Reverse Mortgage Analysis
  • Github
  • PDF Report
  • spreadsheet

An analysis of Level and Indexed annuities, holding a house as security. The goal is to determine the maximum monthly annuity payment which allows debt to the company paying the annuity to not exceed the value of the house.


Sensitivity Analysis: Insurance Products
  • Github
  • Kaggle
  • PDF Report

A sensitivity analysis of two insurance products: a whole life annuity payable in arrears and a whole life assurance. The goal of this project is to compare how sensitive these two products are to interest rate changes by comparing their Expected Present Values(EPVs) for three ages (0, 30 and 60) and interest rates 0% to 50%.


Regression Analysis: Insurance Products

  • PDF Reports

A set of regression analyses exploring different data. Includes: Linear model, Logistic model, Generalised additive model and Mixed effects model
